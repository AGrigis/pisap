This module contains classes for reweighting optimisation implementations

References

1) Candes, Wakin and Boyd, Enhancing Sparsity by Reweighting l1 Minimization, 2007, Journal of Fourier Analysis and Applications, 14(5):877-905. (CWB2007)

class pisap.numerics.reweight.cwbReweight(weights, thresh_factor=1, wtype='image')[source]¶

Candes, Wakin and Boyd reweighting class

This class implements the reweighting scheme described in CWB2007

Parameters:

weights: np.ndarray

Array of weights

thresh_factor: float (optional, default 1)

Threshold factor

wtype: str (optional, default ‘image’)

In which domain the weights are defined: ‘image’ or ‘sparse’.
reweight(data)[source]¶

Reweight.

This method implements the reweighting from section 4 in CWB2007.

Notes

Reweighting implemented as:

w = w (1 / (1 + |x^w|/(n * sigma)))
class pisap.numerics.reweight.mReweight(weights, thresh_factor=1, wtype='image')[source]¶

wtype: str (optional, default ‘image’)

reweight(sigma_est, y)[source]¶

Reweight.

This method implements the reweighting from section 4 in CWB2007.

Notes

Reweighting implemented as:

w = (Ksig * sigma_i) / |Wtx_i| if |Wtx_i| > (Ksig * sigma_i)

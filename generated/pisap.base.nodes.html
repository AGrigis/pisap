class pisap.base.nodes.Node2D(parent, data, node_name)[source]¶

WaveletTransform 2D tree node.

Subnodes are called ‘a’ (LL), ‘h’ (HL), ‘v’ (LH) and ‘d’ (HH), like approximation and detail coefficients in the 2D Discrete Wavelet Transform where:

                            -----------------
                            |       |       |
                            |cA(LL) |cH(HL) |
                            |       |       |
(cA, (cH, cV, cD))  <--->   -----------------
                            |       |       |
                            |cV(LH) |cD(HH) |
                            |       |       |
                            -----------------
HH = 'd'¶
HL = 'h'¶
LH = 'v'¶
LL = 'a'¶

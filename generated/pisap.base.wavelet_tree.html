class pisap.base.wavelet_tree.BaseNode(parent, data, node_name)[source]¶

BaseNode for wavelet transform tree nodes.

The BaseNode is a base class for all nodes. It should not be used directly unless creating a new transformation type. It is included here to document the common node interface and wavelet transform classes.
